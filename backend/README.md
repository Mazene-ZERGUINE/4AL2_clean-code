# Back-end
## Pre-requis
- Node.js

## Installation
```bash
cd backend
npm install
```

## Lancement
```bash
npm run start
```

## Test e2e
- Pre-requis: playwright library 
- installation :
```bash
npm install -g playwright    (install playwright cli globaly)
npx playwright install   (install playwright tools)
```
- running tests: in your termnial use the command
```bash
npx playwright test 
```
